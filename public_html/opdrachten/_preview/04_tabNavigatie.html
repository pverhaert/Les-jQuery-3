Tab navigatie

  • Haal de index van de geklikte link op.
    Gebruik hiervoor de jQuery methode index().
  • Geef de geklikte link de class active.
  • Toon bij opstart enkel het eerste child element (article-tag) binnen #tabs.
    Let op: de pagina bevat ook article-tags die dieper genesteld zijn (descendant elementen)!
  • Link X verbergt de zichtbare article-tag en toont article-tag X.
  • Versie 1: gebruik show/hide.
  • Versie 2: gebruik slideUp/SlideDown
  • Versie 3: gebruik slideUp/SlideDown enkel indien de article-tag niet zichtbaar is.
    (Klikt u bijvoorbeeld 2x op de link Lelie, dan loopt het effect maar eenmaal.)
    Gebruik hiervoor de jQuery methode is()
