Rapport

  • Voeg aan de eerste kolom binnen tbody de class naam toe (tekst wordt groen).
  • Voeg aan de laatste kolom binnen tbody de class score toe (achtergrond wordt lichtgrijs).
  • Geef het cijfer 8/20 en 9/20 een oranje achtergrond (class orange).
    Geef elk cijfer kleiner dan 8/20 een rode achtergrond (class red).
  • Bereken voor elke rij de totaalscore en plaats deze in de laatste kolom.
    Tip: de waarde die je uit een cel ophaalt, is altijd van het type string!
    Zet deze string via de methode parseFloat() om naar een getal.
    Indien de string geen cijfer bevat, is de waarde van parseFloat() gelijk aan NaN (Not a Number).
    Bereken de eindscore enkel voor waardes die niet NaN zijn. Test de waarde met de methode isNaN().
