  • De vier checkboxen bepalen of een foto met een bepaalde class zichtbaar of onzichtbaar staat.
    <input type="checkbox" id="abstract" value="abstract">
  • Merk op dat het id en de value van een checkbox overeenstemt met een class op een foto.
  • Is de checkbox met id #abstract aangevinkt, dan zijn alle li-tags met de class .abstract zichtbaar.
    Is de checkbox met id #abstract afgevinkt, dan zijn alle li-tags met de class .abstract onzichtbaar.
  • Maak de foto's zichbaar/onzichtbaar met de jQuery methodes fadeIn() en fadeOut().
