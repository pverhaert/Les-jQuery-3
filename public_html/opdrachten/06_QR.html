Links met QR-code

  • Bereken voor elke link een korte URL via Google url shortener
  • Voeg de verkorte URL aan de a-tag toe via het data-url attrubuut.
  • Lees met jQuery de short URL uit en genereer de bijbehorende QR-code. Bijvoorbeeld:
    ShortURL Thomas More website: http://goo.gl/sgDj5J
    QR-code Thomas More website: http://goo.gl/sgDj5J.qr (.qr toevoegen aan de URL)
  • De link die de QR-code genereert is een afbeelding. Gebruik deze link als source voor de afbeelding #QRcode.
  • Verplaats de link op de a-tag rond de afbeelding #QRcode.
